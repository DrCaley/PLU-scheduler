# DS-01: Project Overview

**Version**: 0.1  
**Date**: 2025-12-28  
**Status**: Draft

## 1. Purpose

PLU Scheduler is an aid to students and advisors for planning students' next term class schedules along with long-term planning on classes to ensure they fulfill all their requirements for graduation.

## 2. Scope

### 2.1 In Scope

- Term-by-term class schedule planning
- Long-term graduation requirement tracking
- Course requirement visualization
- Student/advisor collaboration features

### 2.2 Out of Scope

- Course registration (handled by existing systems)
- Grade tracking
- Financial aid calculations

## 3. Stakeholders

| Role | Name | Responsibility |
|------|------|----------------|
| Product Owner | Dr. Caley | Requirements approval |
| Primary Users | PLU Students | Schedule planning |
| Secondary Users | Academic Advisors | Guidance and approval |

## 4. Constraints

- Must integrate with PLU's existing course catalog data
- Must handle prerequisite chains and course sequencing
- Academic year/term structure constraints

## 5. Key Concepts

- **Graduation Requirements**: Core, major, minor requirements students must complete
- **Prerequisites**: Courses that must be taken before other courses
- **Course Sequencing**: Optimal order for taking courses
- **Credit Hours**: Unit measurement for course workload
- **Degree Audit**: Tracking completion status of all requirements
